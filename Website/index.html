About

Save & restore your workspace, instantly.

Simple: save every open tab, window, and tab group in one click.

Fast: restore full sessions or inspect them before reopening.

Portable: export/import JSON backups, with optional Cloud Sync when you want it.

How it works

Local first. Sync only when you choose it.

1

Save locally

Manual sessions, Auto Save sessions, tab groups, preview data, and settings are stored in Chrome extension storage on your device.

2

Back up anytime

Export a JSON file for a full portable backup, or import a backup into another Chrome profile when you need to move data manually.

3

Sync optionally

Cloud Sync uses Google Sign-In and the Tabs Session Saver Cloudflare backend. Local changes are pushed automatically after a conservative delay.

Current Cloud Sync limits: up to 10 synced sessions, 300 total URLs, and a 512 KB cloud snapshot. Automatic pushes wait about 10 minutes after changes, and manual Push is rate-limited.
