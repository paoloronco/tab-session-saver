# Tabs Session Saver Data Deletion Instructions

Last updated: July 2026

Tabs Session Saver stores data locally by default. Optional Cloud Sync and newsletter signup may store limited data externally if enabled or submitted by the user.

## Delete Local Extension Data

To delete locally stored sessions and preferences:

1. Open Chrome.
2. Go to `chrome://extensions`.
3. Find Tabs Session Saver.
4. Click "Remove".

Uninstalling the Extension removes locally stored extension data from the browser profile.

Users can also delete individual sessions directly inside the Extension.

## Disable Cloud Sync

To stop syncing on a device:

1. Open Tabs Session Saver.
2. Open Settings.
3. Go to Cloud Sync.
4. Click "Disable".

This disables Cloud Sync on that device. It does not necessarily delete previously synced data from the cloud backend.

## Request Cloud Sync Data Deletion

To delete Cloud Sync data stored on the Tabs Session Saver backend, send a request to:

info@paoloronco.it

Please include the Google account email used for Cloud Sync. We may ask for reasonable verification before deleting cloud data.

## Request Newsletter Data Deletion

To delete newsletter subscription data, send a request to:

info@paoloronco.it

Please include the email address used for newsletter signup.

## Response Time

Deletion requests will be handled as soon as reasonably possible, subject to technical, security, and legal requirements.
